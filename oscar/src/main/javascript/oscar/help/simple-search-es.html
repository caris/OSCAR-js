Búsqueda simple

Simple search provides an entry to searching the catalogue based on the default field, usually csw:AnyText

La búsqueda espacial se provee manteniendo pulsada la tecla CTRL + botón izquierdo del ratón y arrastrando a lo largo de un área.

Búsqueda mejorada

Enhanced searching is available by entering !e into the search field and pressing ENTER or clicking the Search button.

Caracteres comodín

Los caracteres comodín son compatibles con los valores literales.

  • * (Asterisco) - Se utiliza para sustituir múltiples caracteres.
  • . (Punto) - Se utiliza para reemplazar un solo carácter.

Ejemplos

unknown
unkn*n
unkn.wn
