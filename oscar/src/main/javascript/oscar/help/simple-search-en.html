Simple Search

Simple search provides an entry to searching the catalogue based on the default field, usually csw:AnyText

Spatial searching is provided by holding down the CTRL key + Left mouse button and dragging out over an area.

Enhanced Search

Enhanced searching is available by entering !e into the search field and pressing ENTER or clicking the Search button.

Wildcards

Wildcards are supported in literal values.

  • * (Asterisk) - Used to replace multiple characters.
  • . (Period) - Used to replace a single character.

Examples

unknown
unkn*n
unkn.wn
